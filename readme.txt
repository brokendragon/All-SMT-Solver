ALL solvers are built on top of z3-4.3.2-win-x64.

Sources are compiled by Visual Studio 2015.

Experiments are condulted on a laptop with WINDOWS 7.

folder structure:

1. bench: ten industrial benchmarks and five manual crafted benchmarks
2. DPS+CA: source of DPS+CA
3. DPS-child: part source of DPS+CA+SR, should be collaborated with DPS-launcher
4. DPS-launcher: part source of DPS+CA+SR, should be collaborate with DPS-child
5. DPS+SR: source of DPS+SR
6. SPS: source of SPS
7. BCM: source of BCM
8. tcap-1.2.1: source of a command line library we used
